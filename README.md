# crud-usuarios-productos
# Sistema CRUD de Usuarios y Productos

Este proyecto es una aplicación web desarrollada como parte del proceso de aprendizaje en programación.
Permite gestionar usuarios y productos mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

## Tecnologías utilizadas

Frontend:

* React

Backend:

* Node.js
* Express

Base de datos:

* MongoDB

Control de versiones:

* Git
* GitHub

## Estructura del proyecto

```
app_sena_FRONTEND
│
├── app_frontend        # Aplicación frontend desarrollada con React
│
├── app_web_API_085     # API backend desarrollada con Node.js y Express
│
└── .gitignore
```

## Funcionalidades

* Registro de usuarios
* Visualización de usuarios
* Creación de productos
* Visualización de productos
* Edición de productos
* Eliminación de productos

Proyecto desarrollado por estudiante del programa de formación en programación de software.
